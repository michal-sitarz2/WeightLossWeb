Data Privacy

Before setting up an account, please view below on how we use the data you give us.
This website follows GDPR guidelines to protect your data as best as we can!

  • This website collects data from users which are username, password and email. We try to minimize the amount of data we need.
  • We currently do not use your user email for anything, it is just used as an identifier of your account. In the future we might aim to add email reminders with regards to streaks.
  • In addition, you will also need to provide some personal information that we will use to create your progress and your meal plan. Without it the website cannot be used.
  • However, we can ensure all the users that the data given is not used for anything other than generating the meal plan and keeping track of the progress. The data that the users give is safely stored in our database.
  • Nobody outside of the user has access to that data. The data is NOT given to any 3rd parties for any purposes.
  • The only people who have access to the database are developers just for maintaining purposes, and nobody else is given the privileges.
  • For passwords, we are using encryption, so that nobody can view the password, even in the database.
  • We do not have any cookies saving the details, outside of session cookies, and cookies used to keep the user logged in.
  • Outside of the data being stored, when it is transported over the Internet, the best we can do is provide CSRF tokens which minimize the possibility of retrieving that data.
  • Finally, to prevent the website from brute attacks we have enforced a limit on the number of log ins that the user can make, and then the account will be blocked.
